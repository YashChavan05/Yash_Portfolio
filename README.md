# 💼 My Personal Portfolio Website

Welcome to my personal portfolio website — a fully responsive, sleek, and professional showcase of my work, skills, and experience. Designed to impress recruiters, collaborators, and clients from leading tech companies.

> 🚀 Live Preview: [Visit Portfolio](https://yashchavan.vercel.app)

---

## ✨ Features

- 🌓 Light & Dark Mode toggle with smooth transitions
- 🎯 Clean, responsive layout for mobile, tablet & desktop
- 💡 Elegant UI built with modern design principles
- 💬 Animated scroll effects & hover interactions
- 🛠️ Project showcase with live demo & GitHub links
- 📄 Timeline-based Resume & Downloadable CV
- 📩 Contact form with email integration
- 💬 Testimonials & social links
- 💎 Built with a luxury, minimalist color palette

---

## 🛠️ Tech Stack

| Frontend        | Styling        | Animations        | Deployment    |
|-----------------|----------------|-------------------|---------------|
| HTML5 / React / Next.js | Tailwind CSS / SCSS | Framer Motion / GSAP | Vercel / Netlify |
